在Hetzner使用Debian10服务器架设PT盒子的方法
2023/10/8

在Hetzner选择服务器配置CPX21

在配置界面选择随机ssh密码

等待邮箱收到ssh密码信息后

通过ssh软件连接到vps使用jerry048/Dedicated-Seedbox项目的一键脚本

bash <(wget -qO- https://raw.githubusercontent.com/jerry048/Dedicated-Seedbox/main/Install.sh) user password 1024

user和password 替换为自己的信息 这里的用户名和密码为QB的登录账户

使用ssh连接到服务器 选择相应的qb版本 安装bbr分流服务

安装完成后输入reboot重启服务器

qb的登录地址为服务器ip的8080端口

登录qb后设置为中文界面 在高级设置中将磁盘io调整为16

即可开始使用

设置git对github进行SSH传输时候使用443端口
Git 2023/10/8

在用户文件夹的隐藏目录.ssh文件夹创建一个记事本写入以下内容

Host github.com
Hostname ssh.github.com
Port 443

重命名为无扩展名的config

此方法用于解决宽带或代理线路封禁22端口导致git无法正常推送给github的情况
